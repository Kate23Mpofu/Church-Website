# ⛪ Church Website

A modern and responsive mock church website developed as part of my Front-End Developer Internship at Masterpiece Designs.
This project was created to practice building structured, user-friendly websites with clean layouts and responsive design principles.

---

## ✨ Overview

This website was designed to simulate a real church website experience, including sections for:

* Home page
* About section
* Ministries
* Events information
* Contact section
* Responsive navigation

The project focuses on creating a welcoming and accessible user interface while improving front-end development skills and responsive design techniques.
